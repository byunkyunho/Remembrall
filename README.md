# Remembrall
영단어 외우기 웹사이트

버스 기다릴때, 똥 쌀때 수시로 볼라고 만듬

네이버사전 api 이용해서 단어 입력하면 뜻이랑 같이 저장해주고 날짜/철자순 정렬 가능
한글/영어 체크박스로 바꿔가면서 외우기 가능
